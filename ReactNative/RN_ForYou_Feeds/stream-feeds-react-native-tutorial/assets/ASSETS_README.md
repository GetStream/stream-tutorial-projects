# Assets

Add the following PNG files to this directory:

- `icon.png` - App icon (1024x1024 recommended)
- `splash-icon.png` - Splash screen icon
- `adaptive-icon.png` - Android adaptive icon foreground
- `favicon.png` - Web favicon

You can use placeholder images or your own custom icons.

For quick testing, you can download sample Expo assets from:
https://github.com/expo/expo/tree/main/templates/expo-template-default/assets
